# Megan Spielberg - Portfolio

A personal portfolio showcasing my work as a software engineer and game developer, built with React and TypeScript.

## About This Portfolio

This is my professional space where I share both my software engineering case studies and my game development journey. The portfolio is structured to highlight the projects I have worked on during my education, in internships and as side projects.

## What's Included

- **Software Engineering Case Studies** - Public and confidential work framed with appropriate detail levels, including context, role, challenge, and outcome for each project.
- **Game Development Journey** - A timeline of game projects from Bachelor phase through Masters work, showing progression from GDevelop and no-code tools to Godot.
- **Playable Lab** - An interactive clicking game demo showcasing game mechanics and feel.
- **Responsive Design** - Bold visual direction optimized for both desktop and mobile viewing.
- **Media Gallery** - Project screenshots, videos, and YouTube embeds organized in carousels.

## Stack

- **React** - Component-based UI
- **TypeScript** - Type-safe development
- **Vite** - Fast build tooling
- **CSS** - Custom styling with responsive layout

## Project Structure

```text
src/
  components/          # Reusable components (MediaCarousel, GameLab, etc.)
  pages/              # Page components (HomePage, GamePage)
  data.ts             # Case studies and game project data
  types.ts            # TypeScript type definitions
  App.tsx             # Main routing component
  styles.css          # Global styles
  utils.ts            # Helper functions
public/
  media/              # Project screenshots, logos, and videos
```

## Running Locally

1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`
3. Open the local URL shown in terminal (typically `http://localhost:5173`)
